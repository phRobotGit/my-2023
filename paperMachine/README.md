### 项目简介
该项目旨在将 论文（主要为金融领域）生产流程 流水线化。

为达成上诉目标，该项目将
1. 收录常用方法(Methodology)，所有方法接通统一（参考sikit-learn设计）
2. 收录常用数据集，打造数据采集指南
3. 重点关注 ESG, Crypto, Transaction Netwok, Volatility 等话题（Topic）, 提供相应知识图谱
4. 收录常用 词句 (sentence bank) 及 写法 (how to write)

### 项目结构 
paperMachine (主文件夹)
- methodology （收录常用方法）
    - ML

    - sentence_bank
